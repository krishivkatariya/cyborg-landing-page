# CYBORG 2077 – Human × Machine Evolution

A premium, immersive, and highly responsive landing page inspired by futuristic cybernetic human evolution and artificial intelligence integration. Designed with ultra-modern UI details, this page captures the high-tech, glowing sci-fi aesthetic of Cyberpunk 2077 while maintaining fluid performance and smooth user interaction.

## 🚀 Key Features

* **Rich Cyborg Aesthetics**: Dark slate backgrounds combined with vibrant, glowing neon cyan (`#00F5FF`) and neon purple (`#8A2BE2`) accents for an ultra-premium, futuristic look.
* **Neural Network Background**: A custom, high-performance HTML5 Canvas particle system simulating interconnected neural pathways that react to mouse hover.
* **Custom Cybernetic Cursor**: A responsive crosshair tracking system that follows the cursor with smooth lerped delay, changing into a tactical scanning box when hovering over buttons, cards, and inputs.
* **Showcase Laboratory**: An interactive tech-selector panel that allows users to toggle active bionic blueprint upgrades (Synthetic Synapses, Chroma Optics, Nanite Regeneration, Fusion Reactor) to display live diagnostic reports.
* **Animated HUD Dashboards**: Custom metrics (Efficiency, Accuracy, Data connections, Uptime) utilizing dynamic IntersectionObserver count-up script loaders.
* **Neon Timeline Wire**: A vertical milestones path chronicling the chronological steps from the 2026 AI Revolution to the 2077 complete transhumanist singularity.
* **Glassmorphism Panels**: Modern card components built with deep blur backdrop filters and delicate, glowing responsive borders.
* **Fully Responsive HUD**: Custom CSS scaling with media-queries supporting everything from 320px smartphones to standard laptops and 4K ultra-wide monitors.
* **Animate On Scroll (AOS)**: Fluid reveal transitions providing structural flow.

## 🛠️ Tech Stack

* **Structure**: Semantic HTML5
* **Styling**: Modern CSS3 (featuring HSL color variables, Glassmorphism, animations, keyframes, custom web scrollbars, and grid/flex overlays)
* **Logic**: Vanilla ES6 JavaScript (handling canvas renderers, lerping math, observers, form traps, and dynamic UI state toggles)
* **Libraries**: [AOS (Animate On Scroll)](https://github.com/michalsnik/aos) for scroll reveals
